# 3D Mapping

## Definition

3D Mapping reconstructs the geometry and optionally the semantics of an
environment.

## Pipeline

Pose Estimation → Depth → Point Cloud → Fusion → Map Optimization

## Map Representations

-   Point Cloud
-   Occupancy Grid
-   TSDF
-   Octree
-   Neural Fields

## Semantic Mapping

Combine segmentation with geometry for semantic understanding.

## Representative Systems

-   KinectFusion
-   ElasticFusion
-   Voxblox
-   BundleFusion

## Metrics

-   Chamfer Distance
-   Completeness
-   Accuracy
-   Map Consistency

## Datasets

-   Replica
-   ScanNet
-   KITTI-360
-   Matterport3D

## Challenges

-   Loop closure
-   Dynamic environments
-   Real-time processing
-   Memory efficiency
