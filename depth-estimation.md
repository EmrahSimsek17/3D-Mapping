# Depth Estimation

## Definition

Depth estimation predicts the distance between the camera and scene
points.

``` math
Z=fB/d
```

## Categories

-   Monocular
-   Stereo
-   Multi-view
-   RGB-D

## Deep Models

-   Monodepth2
-   ZoeDepth
-   Depth Anything
-   Metric3D

## Metrics

-   Abs Rel
-   Sq Rel
-   RMSE
-   RMSE(log)
-   δ\<1.25

## Datasets

-   KITTI Depth
-   NYUv2
-   ETH3D
-   ScanNet

## Challenges

-   Scale recovery
-   Textureless regions
-   Reflective surfaces
-   Outdoor scenes
